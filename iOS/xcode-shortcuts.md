## Xcode Shortcuts

### Quick auto-indent
`control + i / ⌃ + i`

### Edit all in scope
`command + control + e / ⌘ + ⌃ + e`

### Find next occurrence
`command + control + e / ⌥ + ⌃ + e`

### Find the previous occurrence
`shift + command + control + e / ⇧ + ⌥ + ⌃ + e`

### Move line up or down
Going up:
```
option + command + [ / ⌥ + ⌘ + [
```
Going down:
```
option + command + ] / ⌥ + ⌘ + ]
```
### Multiline cursor
`shift + control + up or down or mouse`

### Go back to where the cursor is
`option + command + L / ⌥ + ⌘ + L`

### Jump to line
`command + L / ⌘ + L`
